1. html tag :- declares the web page to be written in HTML
  2. head tag :- delimits the page's head
  3. title tag :- defines the title
  4. body tag :- delimits the page's body
  5. style tag :- define style of web pages.
  6. script tag :- add javascript in web pages
  7. h1-h6 tag :- delimits a level 1 to 6 heading
  8. p tag :- starts a paragraph
  9. b tag :- make text bold
  10. sup tag :- defines superscript text.
  11. sub tag :- defines subscript text.
  12. u tag :- underline text
  13. br tag :- used for a line break
  14. hr tag :- makes a horizontal line
  15. i tag :- write italic text
  16. small tag :- write smaller text
  17. address tag :- to write address
  18. div tag :- to create a container(block element)
  19. span tag :- to create a container(inline element)
  20. ul tag :- to create a unordered list
  21. ol tag :- To create a ordered list
  22. li tag :- to enter lists item into list
